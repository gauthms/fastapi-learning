A basic To-Do application built with FastAPI that supports CRUD (Create, Read, Update, Delete) operations
Developed as part of a Udemy course: https://www.udemy.com/course/fastapi-the-complete-course/

Deployed live using Render: https://fastapi-todo-application-t61z.onrender.com

To run it locally,
1. Install the dependencies using the following command:
  ```
  pip install -r requirements.txt
  ```
2. Start the application using the following command:
  ```
  uvicorn ToDoApp.main:app --reload
  ```
